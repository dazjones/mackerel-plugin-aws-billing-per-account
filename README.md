# mackerel-plugin-resque

## Usage

```
Usage of mackerel-plugin-aws-billing-per-account:
  -access-key-id string
    	AWS Access Key ID
  -disable-name
    	Disable to get account name. Output account ID.
  -enable-forecast
    	Enable to forecast billing.
  -metric-key-prefix string
    	Metric key prefix. (default "aws-ce")
  -metrics string
    	Choise from [BlendedCost, UnblendedCost, UsageQuantity]. (default "UnblendedCost")
  -secret-access-key string
    	AWS Secret Access Key
  -tempfile string
    	Temp file name
```
